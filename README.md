# Bowling PCB
This project focused on creating a printed circuit board (PCB) for the precise control of a bowling machine, utilizing the KiCAD design platform and the ESP32 microcontroller. The compact and robust design aimed to enhance the accuracy and reliability of ball launches, providing a comprehensive solution compatible with KiCAD standards.
